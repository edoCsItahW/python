### otherColor {#otherColor}
   > 定制其它颜色.
   * Parameters: 
      * param
         * `int r` (Any): 红色通道.
         * `int g` (Any): 绿色通道.
         * `int b` (Any): 蓝色通道.
      * keyword
         * `foreground` (int): 前景色.
         * `colorMode` (int): 颜色模式.

        
### perview {#perview}
   > 展示所有颜色
   
        
### fOtherColor {#fOtherColor}
   > 定制其它颜色.
   * Parameters: 
      * param
         * `_str` (str): 字符串.
         * `_ansi` (str): ANSI转义序列.
      * keyword
         * `RGB` (tuple[int, int, int] | str): RGB值.
         * `foreground` (int): 前景色.
         * `colorMode` (int): 颜色模式.
         * `End` (str): 结束符.

        
### gradient {#gradient}
   > 渐变色.
   * Parameters: 
      * param
         * `_str` (str): 字符串.
         * `beginColor` (tuple[int, int, int]): 开始颜色
         * `endColor` (tuple[int, int, int]): 结束颜色
